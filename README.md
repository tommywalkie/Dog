# Dog
Minimal .NET Core example project powered by C# and F#

Here, the `Dog` (**C#**) is saying hello via `Bark` (**F#**) and will look hungry via `Feel` (**C#**). So we feed the `Dog` using `Feed` (**C#**), making him happy, via `Feel` (**C#**). 

```
   +--------------+ Bark (F#) <--------------+
   |                                         |
   v                                         +
Dog (C#)                                 Feel (C#)
   ^                                         +
   |                                         |
   +--------------+ Feed (C#) <--------------+
```
